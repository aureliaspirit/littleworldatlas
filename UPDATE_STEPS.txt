Little World Atlas v0.2.1 更新步骤

1. 下载并解压 littleworldatlas-v0-2-0-root.zip
2. 打开解压后的文件夹
3. 打开 GitHub 仓库 littleworldatlas
4. 点 Add file → Upload files
5. 把文件夹里面的所有文件拖进去，覆盖旧文件
6. 点 Commit changes
7. 等 1-10 分钟
8. 打开 https://aureliaspirit.github.io/littleworldatlas/

注意：不要上传整个文件夹本身。要上传里面的内容，让 index.html 留在仓库根目录。
