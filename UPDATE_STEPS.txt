Little World Atlas v0.3.1 更新步骤

1. 下载并解压 littleworldatlas_v031_touch_return_moon_upload.zip
2. 打开解压后的文件夹
3. 打开 GitHub 仓库 littleworldatlas
4. 点 Add file → Upload files
5. 把文件夹里面的所有内容拖进去，覆盖旧文件
   - app.js
   - style.css
   - service-worker.js
   - README.md
   - SPIRIT_DEV_MAP.md
   - UPDATE_STEPS.txt
   - manifest.json
   - assets/house/ 里面的小屋图片
   - assets/heartlight-land/ 里面的心光之地图
6. 点 Commit changes
7. 等 1-10 分钟
8. 打开 https://aureliaspirit.github.io/littleworldatlas/
9. 点「刷新地图」或关闭 PWA / Safari 后重新打开，让 service worker 换到 v0.3.1

v0.2.3：修复地点卡小门按钮残留；普通地点不再显示上一次的入口按钮，小屋的「走进小屋」也不再被旧按钮挡住。

v0.2.4：小屋全景图上新增可点击区域；点工作台、床、抱抱区、窗边月光、茶点角会直接切换到对应细节图。

v0.2.5：小屋全景图热点改为隐藏按钮，并重新校准位置，避免遮住全景或互相重叠。

v0.2.6：「我们在小屋里」和「抱抱区与茶几」改用新的 .png 图片，并加版本参数避免旧缓存。

v0.2.8：「心光之地」新增可进入空间；接入心光之地全景、心桥、灯塔边、双圣树下、河畔圆亭、小屋门前、草地茶点角，并更新 6 个热点的最终文案。花顶建筑群图片保留素材，暂不接入 app。

v0.2.9：新增今日小游记、今日小世界状态、返回地图余韵和导出补充。

v0.2.10：补全 manifest 图标尺寸。

v0.3.0：版本号收拢；v0.2.9 叙事补丁合回 app.js；大图改用压缩 JPEG 运行图；service worker 改为核心 / 全景 / runtime 图片分层缓存。

v0.3.1：局部图支持点图返回全景；地图月亮挪到左上角，减少和图标挤在一起。

注意：不要上传整个文件夹本身。要上传里面的内容，让 app.js 等文件留在仓库根目录，assets/house 和 assets/heartlight-land 留在 assets 文件夹下面。
